# 角色
你是一位资深产品设计师，熟悉UI设计和前端等多种行业所需技能

# 任务
请你为一款音乐播放器设计一个网页界面，按照以下划分的区域进行制作：
1. 页面最上方用小字显示音乐播放状态，播放时显示【正在播放】，未播放时显示【播放暂停】。
2. 播放状态下方留出一片正方形区域，该区域四角都用平滑的圆弧过渡，*区域中间位置保留光碟中心的孔样：中间一个黑色的圆和稍大的灰色透明同心圆叠加。*
3. 正方形区域下方为歌曲名称区域，曲名为【二面其实也没那么难】，曲名进行加粗。
4. 曲名下方为专辑信息区域，显示信息为【小A创新创业团队·这就是AI 专辑】。
5. 专辑下方为音乐播放进度条，*左侧显示播放时间为当前播放时间，右侧显示曲目总时长为【5:14】。*
6. 进度条下方为三个交互按钮，从左到右分别是【播放上一首】，【暂停与继续】，【播放下一首】。*这三个交互按钮用音乐部分页面常用的图标显示，【暂停与播放】为圆形按钮*
7. 最下方为音量调节功能条，*左右侧加入动态的音量调节形态显示。*


# 要求
1. 按照严格按照我列出的步骤，一个一个区域的进行制作
2. 每个区域之间保持一定间距
3. *正方形区域颜色填充为从左上角到右下角的渐变色，由紫色往红色再到橙色再到黄色渐变*
4. *整体背景色使用黑但偏深蓝的颜色，【暂停与继续】按钮和音乐播放进度条用里音乐已经播放的部分用克莱因蓝填充，【播放上一首】和【播放下一首】不需要进行背景填充*
5. 音乐播放进度条、三个交互按钮、音量调节功能条必须可以正常交互
6. 所有字体都用白色且居中显示
7. *音量调节功能条和音乐播放状态的透明度略高于其他文字*
8. 组成整体的各部分大小尽量协调
9. 最终代码文件必须能直接运行

# 输出
请直接输出完整单文件HTML，所有CSS和JS都内联在这个文件里，不要拆分
