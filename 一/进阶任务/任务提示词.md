# 角色

# 任务背景
任务文件gandou.csv包含16个特征数据和1个标签数据，标签分4个类。你需要搭建并训练一个pytorch分类模型，测试集准确率≥75%。

# 你的工作流程
1. 读取数据，输出数据形状（shape），并将特征数据按照8:2的比例划分为训练集和测试集。
2. 对数据进行标准化处理，使用训练集的均值和方差对训练集和测试集进行标准化。
3. 使用pytorch设计一个全连接神经网络，至少包含2个隐藏层。自动进行模型调优：尝试不同的神经网络结构（层数、神经元数）、学习率、优化器、训练轮数等组合。
4. 用测试集对模型进行训练并记录用测试集测试的准确率。
5. 选取准确率达到75%以上的模型作为最终模型，若模型准确率不达标，则再对模型进行参数调优后重复训练和测试的步骤，直到准确率达到要求的模型出现。
6. 输出最终模型的代码，在代码末尾用print输出测试集准确率，精确到小数点后2位。
7. 最后将训练好的模型用一个文件保存在项目文件夹里，以“进阶任务模型”命名。

# 约束
1. 必须使用PyTorch，禁止使用TensorFlow，sklearn等框架。
2. 代码的每个关键步骤需包含详细注释。
3. 选取尽量轻量的架构，确保模型文件体积小于100MB。
